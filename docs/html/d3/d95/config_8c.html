char * getProperty (char **file, int lines, const char *property, int lenght, int *isAList, int *listLen)
 Ottenimento di una proprietà dal file di configurazione. La funzione cerca linearmente la proprietà richiesta all'interno del file letto, e, nel caso la proprietà venga trovata, restituisce un puntatore alla stringa definita dalla proprietà nel file. Nel caso la proprietà richiesta sia una lista, viene restituito un puntatore a char con valore uguale alla lunghezza della lista, e l'intero isAList verrà settato a 1. Continua...
